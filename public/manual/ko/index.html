앱 소개

GTR 트레이닝(저장소명 puyo-simulator)은 뿌요 계열 낙하 퍼즐의 대표적인 정형 연쇄인 GTR과 계단을 AI 추천과 함께 연습하는 웹 애플리케이션입니다.

  • AI는 ama(MIT, citrus610 제작)를 WebAssembly / 네이티브로 내장합니다.
  • 웹 기반 PWA — 설치 후 오프라인 실행 가능.
  • 템플릿은 AI의 가중치를 조정하여 GTR 또는 계단 형태에 맞게 추천합니다.
  • 세 가지 모드: 자유 연습, ama 점수 대결, 점수 어택.

키동작

터치(모바일)

  • 보드를 좌우로 스와이프 → 좌우 이동
  • 보드를 아래로 스와이프 → 소프트 드롭
  • 보드를 탭 → 회전(탭 위치에 따라 CW / CCW)
  • 화면 하단 버튼으로도 동일 조작 가능

화면 하단 버튼

  • Drop(↓) — 현재 열·회전으로 확정.
  • ★ AI Best — ama의 최선수를 즉시 실행. 반복 누르면 AI 자동 진행.
  • Undo(↶) — 한 수 되돌리기(매치 중에는 자기 쪽만).
  • Reset — 보드와 시드 리셋. 확인 다이얼로그 표시.

게임 모드

헤더 좌측 모드 셀렉터로 세 가지 모드를 전환합니다.

1. 자유(Free)

무제한 연습 모드. AI 후보가 보입니다. 후보를 보고 수를 고르고, 가끔 AI Best로 정답을 확인하고, 맞지 않으면 Undo로 다른 수를 시도하는 식의 반복 연습에 적합합니다.

2. ama 점수 대결(Score vs ama)

양쪽이 같은 시드(동일한 NEXT)로 정해진 수(30 / 50 / 100)를 두며 누계 점수가 높은 쪽이 이깁니다. AI 후보는 표시되지 않아 순수한 수읽기 대결이 됩니다.

  • 오른쪽 Resign 버튼으로 도중 종료 가능.
  • 매치 종료 후 메뉴의 「저장된 매치」에 보관할 수 있습니다.

3. 점수 어택(Score attack)

상대 없는 단독 모드. 서버에 업로드하면 짧은 URL로 리플레이를 공유 가능(?score=<id>로 로드).

템플릿(트레이너)

  • 없음 — 기본 빔 서치(build 프리셋).
  • GTR — GTR의 키와 접는 부분에 가중치를 부여하여 GTR 형태로 유도.
  • 계단 — 계단 연쇄의 토대를 우선시.

템플릿 변경은 자유 모드의 후보와 대전 시 ama의 플레이 스타일에 함께 반영됩니다.

편집 모드

  • 헤더의 「Edit」 버튼으로 진입(대전 중이면 종료 확인 다이얼로그가 나옵니다).
  • 하단 팔레트의 R / P / B / Y에서 색을 고르고 셀을 탭해 배치.
  • ○는 지우기, ✕는 방해 뿌요 배치.
  • 오른쪽 Current / NEXT / NEXT 2 카드를 탭하면 현재·다음 페어를 편집할 수 있습니다.
  • Apply 적용, Cancel 취소, Clear 보드 전부 지우기.
편집한 보드는 메뉴의 공유 기능으로 URL에 인코딩하여 다른 사람에게 보낼 수 있어 연쇄 퍼즐 출제에 편리합니다.

메뉴와 표시 설정

  • Ghost(고스트) — 현재 페어가 떨어질 위치를 흐리게 미리보기.
  • Ceiling(천장) — 윗줄 질식 라인 표시 토글.
  • Template(템플릿) — AI 가중치(위 참고).
  • Language(언어) — UI 언어를 한국어 / 일본어 / 영어 / 중국어로 전환. 매뉴얼도 함께 적용됩니다.
  • Share(공유) — 현재 보드를 URL에 인코딩.
  • Saved matches(저장된 매치) — 저장된 리플레이 목록.
  • 매뉴얼 — 이 매뉴얼을 새 탭에서 엽니다.
  • Analyze(분석) — 자유 / 종료된 매치의 자기 수를 ama가 재평가하여 일치율과 평가값(%)을 산출.

공유 · 저장 · 분석

보드 공유

URL을 다른 기기에서 열면 같은 보드가 재현됩니다. 점수와 히스토리는 전달되지 않습니다.

리플레이 공유

점수 어택 / 매치 종료 후의 「Share replay」는 시드와 수순을 URL에 인코딩합니다. 서버 업로드 후 짧은 ID URL을 받을 수도 있습니다(?score=<id>로 로드).

저장된 매치

시드와 수순만 저장되므로 용량은 매우 작습니다. 리플레이 중에는 턴 단위로 이동하며 연쇄 애니메이션을 다시 볼 수 있습니다.

분석(Analyze)

  • AI 일치율 — 자기 수가 ama의 최선수와 일치한 비율(AI Best 버튼으로 둔 수는 제외).
  • AI 평가값 — 자기 수의 점수가 ama 최선수 대비 몇 %인지 평균(ama 상위 5수 안에 든 수만 집계).

PWA / 네이티브 앱

이 앱은 PWA로 동작합니다. Chrome / Safari에서 "홈 화면에 추가"하면 앱처럼 사용하고 오프라인 실행도 가능합니다.

macOS / Android에는 Tauri 네이티브 빌드도 있습니다. ama를 Rust FFI를 통해 C++ 정적 라이브러리로 호출하므로 AI 응답이 Intel Mac에서 p99 200ms 이내까지 단축됩니다.

용어

축 뿌요페어 회전의 중심이 되는 뿌요. 보드에서 빨간 테두리로 표시됩니다.
연쇄같은 색 4개 이상이 사라지면서 위 뿌요가 떨어져 다시 사라지는 반응.
GTR뿌요뿌요 통 계열의 대표 정형. L자 키 + 접는 형태.
계단색을 계단 모양으로 배치해 만드는 연쇄 정형.
질식13행 이상에 뿌요가 놓여 게임 오버되는 상태.
seedNEXT를 생성하는 난수 시드. 공유하면 같은 순서를 재현 가능.
ama본 앱에 내장된 AI(citrus610 제작, MIT). 빔 서치 + 휴리스틱.

「ぷよ / Puyo」는 SEGA / Compile의 등록 상표입니다. 본 앱은 연습용 시뮬레이터이며 상업 배포 전에는 리브랜드가 필요합니다. 자세한 내용은 저장소 README.md 참고.
