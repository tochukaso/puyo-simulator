应用简介

GTR 训练(仓库名 puyo-simulator)是一款练习 GTR 与 阶梯 等经典连锁形的网页应用, AI 顾问每回合给出最优手与前 5 候选。

  • AI 引擎为 ama(MIT 协议,作者 citrus610),通过 WebAssembly / 原生静态库加载。
  • 纯网页应用,可作为 PWA 安装并离线启动。
  • 模板会调整 AI 权重,使建议偏向 GTR 或阶梯的组形。
  • 三种游玩模式:自由练习、对 ama 计分、计分挑战。

底部按钮

  • Drop(↓) — 在当前列与朝向落下。
  • ★ AI Best — 立即执行 ama 推荐手,反复点击则 AI 自走。
  • Undo(↶) — 撤销一步(对战中仅作用于自方)。
  • Reset — 重置盘面与随机种子,会出现确认对话框。

游戏模式

顶栏左侧的模式选择器切换三种模式。

1. 自由(Free)

无手数限制、显示 AI 候选,适合学习与摸索。看着候选选择手,偶尔参考 AI Best,不合适就 Undo 重来。

2. 对 ama 计分对决(Score vs ama)

双方使用相同种子(NEXT 完全一致),在固定手数(30 / 50 / 100)内打完累计分数高者获胜。 AI 候选不显示 — 完全比较手势选择能力。

  • 右上的 Resign(投降)按钮可中途结束。
  • 对战结束后从菜单的「保存对战」可保留回放。

3. 计分挑战(Score attack)

无对手的单人模式,可上传到分数服务器获取短链接(用 ?score=<id> 加载回放)。

模板(训练器)

  • 无 — 默认 beam search(build 预设)。
  • GTR — 调高 GTR 钥匙与折返列的权重,使建议偏向 GTR。
  • 阶梯 — 优先阶梯连锁底盘的权重。

切换模板会同时影响自由模式的 AI 候选与对 ama 模式中 ama 自身的下法。

编辑模式

  • 顶栏的「Edit」按钮进入编辑(对战中会先确认中断对战)。
  • 底部 R / P / B / Y 选颜色,点击格子放置。
  • ○ 清除(变空格),✕ 放置干扰气泡。
  • 右侧 Current / NEXT / NEXT 2 卡片可点击修改当前对与下两对。
  • Apply 提交,Cancel 取消,Clear 全部清空。
编辑后的盘面可通过菜单的分享编码成 URL,适合发布连锁谜题。

菜单与显示设置

  • Ghost(幽灵预览) — 显示当前对将落下的虚影。
  • Ceiling(顶部) — 切换窒息线一行的视觉显示。
  • Template(模板) — AI 权重(见上文)。
  • Language(语言) — 切换 UI 语言:日本語 / English / 中文 / 한국어。本手册随之联动。
  • Share(分享) — 把当前盘面编码到 URL。
  • Saved matches(保存的对战) — 已保存的对战回放列表。
  • 使用手册 — 在新标签页打开本手册。
  • Analyze(解析) — 让 ama 重新评估自由 / 已结束对战的每一手,给出 AI 一致率与 AI 评估值。

分享 / 保存 / 解析

盘面分享

对方打开此 URL 可复现相同盘面。分数与历史不会带过去。

回放分享

计分挑战 / 对战结束后的「Share replay」会把 种子与手序 编码到 URL。 也可上传到服务器换取短链接,通过 ?score=<id> 直接加载。

已保存对战

仅保存种子与手序,体积极小。回放时可逐回合切换、复看连锁动画。

解析

  • AI 一致率 — 自手与 ama 最优手一致的比例(通过 AI Best 按钮打的不计入)。
  • AI 评估 — 自手得分相对 ama 最优手的百分比平均(仅计入进入 ama 前 5 候选的手)。

PWA / 原生版

本应用是 PWA,在 Chrome / Safari 中可「添加到主屏幕」并离线启动。

macOS / Android 还提供 Tauri 原生版本,通过 Rust FFI 调用 C++ 静态库形式的 ama, 在 Intel Mac 上 AI 思考 p99 可控制在 200ms 以内。

术语表

轴气泡对中作为旋转中心的气泡,在盘面上以红框标记。
连锁消除导致上方掉落,继续连续消除的反应。
GTR魔法气泡通系列代表性的定型,L 形钥匙 + 折返。
阶梯颜色按阶梯排列形成的连锁定型。
窒息气泡到达第 13 行游戏结束。
seed(种子)生成 NEXT 序列的随机数种子,共享后可还原相同序列。
ama本应用内置的 AI(citrus610 制,MIT),beam search + 启发式。

「魔法气泡 / ぷよ / Puyo」是 SEGA / Compile 的注册商标。本应用为练习用模拟器,商用前请改名。详情见仓库 README.md。
